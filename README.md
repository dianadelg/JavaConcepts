# JavaConcepts

A comprehensive repository housing Java implementations of fundamental data structures, algorithmic problem-solving, and Object-Oriented Programming (OOP) design patterns.

## Repository Structure

The project is organized into several key modules:

* **`data_structures`**: Custom implementations of standard data structures to understand their underlying mechanics. Additionally, solved HackerRank and Leetcode problems
* **`intro_java`**: Foundational concepts and introductory exercises.
* **`studying_2026`**: Efficient solutions to various, recent LeetCode challenges, focusing on optimized time and space complexity. Additional resources stumbled upon while strengthening my skills. NOTE: This folder is more up to date than the leetcode subfolder in data_structures

---

## Key Components

### 1. Data Structures
This module explores the implementation and usage of:
- **Lists**: Singly and Doubly Linked Lists.
- **Stacks & Queues**: Array-based and Link-based implementations.
- **Trees**: Binary Search Trees (BST) and traversal algorithms.
- **Sorting & Searching**: Implementation of Merge Sort, Quick Sort, and Binary Search.

### 2. LeetCode Solutions
A collection of problem-solving scripts categorized by difficulty and topic, including but not limited to:
- **Arrays & Hashing**
- **Two Pointers**
- **Sliding Window**
- **Dynamic Programming**

### 3. Photo Album Simulation (OOP)
A deep dive into **Object-Oriented Programming** principles. This sub-project simulates a photo album management system utilizing:
- **Encapsulation**: Protecting image metadata and album state.
- **Inheritance**: Creating specialized album types.
- **Polymorphism**: Implementing flexible search and sorting methods for photos.
- **Abstraction**: Defining clear interfaces for album operations.

## Getting Started

### Prerequisites
- **JDK 11** or higher.
- A Java IDE (IntelliJ IDEA, Eclipse, or VS Code).

### Installation & Running
1. Clone the repository:
   ```bash
   git clone [https://github.com/dianadelg/javaconcepts.git](https://github.com/dianadelg/javaconcepts.git)

2. Navigate to the desired package
   ```bash
   cd javaconcepts/data_structures

3. Compile and run any specific class
   ```bash
   javac FileName.java
   java FileName
